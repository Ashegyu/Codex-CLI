CLI Chat 사용 설명서

Codex CLI 기반 채팅 앱 기본 사용 가이드

1. 기본 사용

  • 하단 입력창에 질문을 입력하고 Enter로 전송합니다.
  • Shift+Enter는 줄바꿈입니다.
  • 좌측 상단의 새 대화 버튼으로 새 세션을 시작합니다.

2. 스트리밍 응답 화면

  • 답변 탭: 최종 응답 표시
  • 과정 탭: 작업 단계/명령 진행 로그
  • 코드 탭: 변경 파일/코드 스니펫/Diff

진행 로그 패널은 스크롤 가능하며 이전 단계 이력을 확인할 수 있습니다.

3. 자주 쓰는 명령

  • /help: 슬래시 명령 목록
  • /resume [session-id]: 기존 Codex 세션 이어서 실행
  • /status: 5h/weekly limit 갱신
  • /model, /reasoning, /sandbox: 런타임 옵션 변경

4. 파일 열기

  • 답변 내 파일 링크를 클릭하면 로컬 파일을 기본 앱으로 엽니다.
  • @경로 또는 /file [경로]로 파일 내용을 입력창으로 가져올 수 있고, 파일 선택 창에서는 여러 파일을 한 번에 고를 수 있습니다.

5. 단축키

  • Ctrl+N: 새 대화
  • Ctrl+B: 좌측 패널 접기/펼치기
  • Esc: 실행 중 작업 중지
